# 餘論 蘋果日報的史料價值

《蘋果日報》係由黎智英所創辦，因其腥羶色、反共鮮明的政治立場聞名。自 2000 年代初，蘋果日報來到台灣（又稱「臺蘋」）後逐漸超越台灣的三大報，一度改變台灣的報業生態。
